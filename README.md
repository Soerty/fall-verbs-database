# База данных глаголов падения русского языка

Странная штука.


### Деплой

Очень надеюсь, что у вас Linux, иначе разбирайтесь сами:
```Bash
$ sudo pip -r requirements
$ python3 fall-verbs/manage.py runserver
```


### Как добавлять слова

Через панель администратора:
```
login: admin
passw: super-admin
```
